## Консольное приложение для выполнения операций над клиентами и счетами.
## Вариант реализации 1

Импортируйте проект в свою IDE, запустите метод main.

В данном приложении подразумевается, что:
1. Пользователь приложения вводит все данные корректно и валидация не требуется.
2. ФИО клиента является уникальным неповторящимся значением.
